## Color Usage Examples

### Backgrounds
- **Page background**: `bg-background` (#1a1a2e)
- **Card background**: `bg-card` (#16213e)
- **Card hover**: `bg-card-hover` (#1f3056)
- **Surface elements**: `bg-surface` (#0f3460)

### Text
- **Primary text**: `text-gray-200` (#E0E0E0) or `text-white`
- **Secondary text**: `text-gray-400` (#A0A0A0)
- **Muted text**: `text-gray-500`

### Accents
- **Primary accent**: `text-accent` or `bg-accent` (#FF9200)
- **Accent hover**: `hover:bg-accent-hover` (#FFB600)
- **Deal price**: `text-price-deal` (#FF4444)
- **Original price**: `text-price-original` (#888888)
- **Discount badge**: `bg-price-discount` (#FF6B35)

### Borders
- **Default border**: `border-border` (#2a2a4a)
- **Accent border**: `border-accent` or `border-accent/30` (with opacity)

## Responsive Behavior

### Grid Columns
- **Mobile (< 640px)**: 1 column
- **Tablet (640px+)**: 2 columns
- **Desktop (1024px+)**: 3 columns
- **Large (1280px+)**: 4 columns

### Navigation
- **Mobile**: Hamburger menu (placeholder), expandable search
- **Desktop**: Full navigation bar, search always visible

### Hero Section
- **Mobile**: Trending keywords hidden
- **Desktop**: Trending keywords displayed inline

## Accessibility Features

- Semantic HTML (`<header>`, `<main>`, `<article>`, `<nav>`)
- ARIA labels on buttons and inputs
- `alt` text on all images
- Keyboard navigation support
- Focus states on interactive elements
- Color contrast meets WCAG AA standards
- Skip to main content (future enhancement)

## Animation Examples

### Fade In
```css
@keyframes fadeIn {
  0%   { opacity: 0 }
  100% { opacity: 1 }
}
/* Usage: animate-fade-in */
```

### Slide Up
```css
@keyframes slideUp {
  0%   { transform: translateY(10px); opacity: 0 }
  100% { transform: translateY(0); opacity: 1 }
}
/* Usage: animate-slide-up */
```

### Pulse (Skeleton)
```css
@keyframes pulse {
  0%, 100% { opacity: 1 }
  50%      { opacity: 0.5 }
}
/* Usage: animate-pulse */
```

## Design Inspiration

Based on QuasarZone's deal board (https://quasarzone.com/bbs/qb_saleinfo):
- Dark navy background with orange accents
- Clean card-based layout
- Prominent price display
- AI score indicator (unique to DealHawk)
- Community engagement stats (views, votes, comments)
- Modern, minimal aesthetic
